# Interactive Wizard

The MCP Generator includes an interactive wizard for guided configuration. The wizard activates when you run `stellar mcp generate` with no flags.

## Starting the Wizard

```bash
stellar mcp generate
```

## Wizard Flow

The wizard guides you through 7 configuration steps:

<Steps>

### Contract ID

Enter the contract ID of your deployed Soroban contract.

```
🚀 Enter the contract ID (starts with 'C', 56 characters):
> CBX7HKQ6WIOYAHS6SLL7Y3MRPTYOSGJTDL6XXYYOTEWEN3PRGK3DSVZG
```

**Validation:**
- Must be exactly 56 characters
- Must start with 'C'
- Must be valid base32 encoding

### Network Selection

Select the network where your contract is deployed.

```
🌐 Select the network:
> testnet
  mainnet
  futurenet
  local
  custom
```

If you select **custom**, you'll be prompted for:
- Custom RPC URL
- Network passphrase

### Language Selection

Choose your target language.

```
💻 Select the target language:
> typescript
  python
```

### Frontend Generation

Choose whether to generate a web frontend.

```
⚙️ Generate a React frontend alongside the MCP server?
> No - MCP server only
  Yes - Generate React frontend
```

### Output Directory

Specify where to generate the project.

```
📁 Enter output directory:
> ./my-token-mcp
```

**Default:** `./mcp-server`

### Contract Name (Optional)

Optionally provide a name for your contract.

```
📝 Enter contract name (optional, press Enter to skip):
> my-token
```

If skipped, the generator attempts to:
1. Read the name from contract metadata
2. Use the first 8 characters of the contract ID

### MCP Server Name

Set the name for MCP server registration.

```
🏷️ Enter MCP server name:
> soroban-contract
```

**Default:** `soroban-contract`

</Steps>

## Visual Features

The wizard uses color-coded output:

| Color | Purpose |
|-------|---------|
| **Cyan** | Headers and section titles |
| **Green** | Success messages and confirmations |
| **Yellow** | Warnings and important notes |
| **Red** | Errors and validation failures |

### Emoji Support

The wizard displays emojis with ASCII fallbacks:

| Emoji | Fallback | Usage |
|-------|----------|-------|
| 🚀 | `>` | Start/action prompts |
| ✨ | `*` | Success indicators |
| 📁 | `[]` | Directory/file references |
| ⚙️ | `[*]` | Configuration options |
| ✅ | `[OK]` | Completed steps |
| ℹ️ | `i` | Information |

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate options |
| `Enter` | Select / Confirm |
| `Ctrl+C` | Cancel wizard |
| `Backspace` | Clear input |

## When to Use Wizard vs Flags

| Scenario | Recommended |
|----------|-------------|
| First time using the generator | Wizard |
| Exploring options | Wizard |
| CI/CD automation | Flags |
| Scripted generation | Flags |
| Quick iteration | Flags |

## Disabling Wizard

To always use flags mode, provide at least the required `--contract-id`:

```bash
# This will NOT trigger the wizard
stellar mcp generate --contract-id CABC...

# This WILL trigger the wizard
stellar mcp generate
```
