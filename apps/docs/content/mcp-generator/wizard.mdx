# Interactive Wizard

The MCP Generator includes an interactive wizard for guided configuration. The wizard activates when you run `stellar mcp generate` with no flags.

## Starting the Wizard

```bash
stellar mcp generate
```

## Wizard Flow

The wizard guides you through 6 configuration steps:

<Steps>

### Contract ID

**Validation:**
- Must be exactly 56 characters
- Must start with 'C'
- Must be valid base32 encoding

### Network Selection

```

If you select **custom**, you'll be prompted for:
- Custom RPC URL
- Network passphrase

```

### Output Directory

Specify where to generate the project.

```

```

**Default:** `./mcp-server`

### Contract Name (Optional)

Optionally provide a name for your contract.

```

```

If skipped, the generator attempts to:
1. Read the name from contract metadata
2. Use the first 8 characters of the contract ID

## When to Use Wizard vs Flags

| Scenario | Recommended |
|----------|-------------|
| First time using the generator | Wizard |
| Exploring options | Wizard |
| CI/CD automation | Flags |
| Scripted generation | Flags |
| Quick iteration | Flags |

## Disabling Wizard

To always use flags mode, provide at least the required `--contract-id`:

```bash
# This will NOT trigger the wizard
stellar mcp generate --contract-id CABC...

# This WILL trigger the wizard
stellar mcp generate
```
