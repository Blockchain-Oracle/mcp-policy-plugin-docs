## Try It Yourself

Generate your own projects:

```bash
# TypeScript MCP Server
stellar mcp generate \
    --contract-id CAHLJEQUCNTV7JPAPCMLCBIHOX7FFB57DUARJ6XGTW27FPCVKKY7JM2A \
    --network testnet \
    --lang typescript \
    --output my-token-mcp

# Python MCP Server
stellar mcp generate \
    --contract-id CAHLJEQUCNTV7JPAPCMLCBIHOX7FFB57DUARJ6XGTW27FPCVKKY7JM2A \
    --network testnet \
    --lang python \
    --output my-token-mcp-python

# Policy Contract (interactive)
stellar policy generate
```
