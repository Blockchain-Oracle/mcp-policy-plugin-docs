# Getting Started

This guide walks you through installing and setting up the Stellar MCP Plugins on your development machine.

## What You'll Set Up

By the end of this guide, you'll have:

1. **stellar-mcp-generator** - Generate MCP servers from Soroban contracts
2. **stellar-policy-cli** - Create policy-enforced smart contracts via wizard

Both plugins integrate with the Stellar CLI and extend it with new subcommands.

## Time Required

- **Prerequisites**: 5-10 minutes (if you need to install dependencies)
- **Plugin Installation**: 2-3 minutes each
- **Verification**: 1 minute
